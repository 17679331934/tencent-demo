## 问题1：
  请编写一个函数 twoSum(nums, target)，其中 nums 是一个整数数组，target 是一个整数目标值，函数应该返回一个数组，其中包含两个数字的索引，这两个数字的和等于 target。

  示例：
  输入：nums = [2, 7, 11, 15], target = 9
  输出：[0, 1]
  解释：因为 nums[0] + nums[1] = 2 + 7 = 9，所以返回 [0, 1]。

  用JavaScript的Alert显示结果即可
